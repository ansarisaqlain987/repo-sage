# Repo Sage Name Generator

 The Repo Sage Name Generator is a web-based application that assists developers in creating unique and descriptive names for their Git repositories. It aims to enhance productivity by alleviating the common challenge of coming up with suitable repository names

## Technologies used

- [NextJS 14](https://nextjs.org/)
- [React 18](https://react.dev/)
- [Typescript](https://www.typescriptlang.org/)
- [Bun 1.1.30](https://bun.sh/)
- [Shadcn UI](https://ui.shadcn.com/)
- [Shadcn Color Theme](https://ui.jln.dev/)
